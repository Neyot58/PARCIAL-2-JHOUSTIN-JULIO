# PARCIAL-2-JHOUSTIN-JULIO

private string fechaVencimiento; es un atributo de la clase ProductoAlimento que se usa para almacenar la fecha en que un alimento vence.

La palabra private indica que la variable solo puede ser utilizada dentro de la misma clase, lo que ayuda a proteger los datos y aplicar encapsulación en la programación orientada a objetos.

El tipo de dato string se utiliza porque la fecha se está guardando como texto (por ejemplo: "20/03/2026").

Este atributo permite que cada objeto de tipo ProductoAlimento tenga su propia fecha de vencimiento, diferenciándolo de otros tipos de productos como los electrónicos. Normalmente se accede o modifica mediante propiedades (get y set) o métodos dentro de la clase.
